The Project

This site explores the results of a weather plotting project using Python and Jupyter Notebooks. The project's goal is to provide analytic proof of basic weather trends by comparing cloudiness, wind speed, temperature, and humidity to the latitude of cities in a random sample. The sample was obtained by using the nearest_city method in the CitiPy library to generate a list of 500+ cities from randomly generated latitude and longitude coordinates using the random library. Those coordinates were then used to query the OpenWeather API for weather data. Using Pandas, Numpy, and SciPy, the results were transformed into dataframes and plotted using Matplotlib. All plots are timestamped with the date and time of the Open Weather API request.

This site was built with HTML5, external CSS, and BootstrapCDN.
